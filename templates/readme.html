关于 LM Chat

LM Chat 是一个基于本地大语言模型的对话系统，旨在为用户提供 情绪支持、心理教育与日常交流。 本系统并非医疗或心理治疗工具，不提供诊断或处方。

系统说明

  • 本网站主要用于学习、研究与内部测试用途。
  • 对话内容由本地或服务器端模型生成，可能存在不完整或不准确之处。
  • 请勿将系统回复作为专业医疗、心理或法律建议。

安全措施、输入输出与使用限制说明

本站为本地部署的实验性对话系统，主要用于教学展示、技术验证及有限场景下的辅助交流。 为保障使用安全、合规性与校园环境稳定，系统已实施以下限制与防护措施：

一、使用场景限制

  • 本站仅限在校内网络 / 校内终端（如校园电子屏）环境中使用。
  • 不作为商业产品或公众服务平台，不适用于紧急情况处理或高风险决策支持。

二、输入内容说明

  • 用户输入内容为自然语言文本，用于一般性交流、学习讨论或情绪表达。
  • 请勿输入真实姓名、联系方式、身份证号等敏感个人信息。
  • 系统不鼓励、也不支持任何违法、危险或伤害自身与他人的内容。
  • 当输入内容触发系统设定的高风险关键词或模式时，可能被限制处理。

三、输出内容说明

  • 系统输出由语言模型自动生成，仅作为参考性文本。
  • 输出内容不构成医疗、心理诊断、法律意见或任何专业结论。
  • 在风险场景下，系统可能仅返回安抚性、提示性或引导性回复。
  • 部分输出内容可能经过裁剪、重写或替换，以符合安全策略。

四、人工审阅与干预机制

  • 在满足特定条件时，对话可能进入人工审阅流程，用于风险评估与教学管理。
  • 教师端仅用于风险标记与必要提示，不进行持续性实时监控。

五、账号与登录状态说明

  • 账号系统仅用于区分使用者与对话记录，数据存储于本地服务器。
  • 请在使用完毕后主动点击“退出登录”，尤其是在公共终端或电子屏环境下。

六、责任声明

  • 系统回复不代表任何个人、教师或学校官方立场。
  • 使用者需对自身行为与决策负责。
  • 如遇严重心理困扰或紧急风险，请立即联系校内老师或专业机构。

本说明可能随系统版本或安全策略调整而更新。

如果你不同意以上说明，请不要继续使用本系统。
